Infamous Second Sons

Infamous Second Sons is a seque to the infamous fanchise That started in 2009. Now personal this is a game i will always recommend to newcomers of the fanchise.

Now Infamous Second Son is an action-adventure game set in an open world environment and played from a third-person perspective. Players control the main character Delsin Rowe, who can parkour-style climb vertical surfaces like high-rise buildings. Delsin is a Conduit, which allows him to use superpower abilities by manipulating materials such as smoke, neon, video, and concrete.
